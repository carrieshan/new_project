# 数据库监控系统

这是一个用于监控数据库状态的综合工具，包含数据库健康检查、SQL查询、异常检测、定时任务和邮件通知功能。

## 功能特性

### 1. 数据库健康检查
- 检查表是否存在
- 检查表的行数
- 检查表的列信息
- 检查外键约束
- 检查数据库完整性
- 生成详细的检查报告

### 2. SQL查询功能
- 实时SQL执行
- 结果集展示
- 查询历史记录
- 查询性能统计

### 3. 异常检测
- 异常SQL查询检测
- 阈值设置
- 报警机制

### 4. 定时任务
- Cron表达式支持
- 任务调度管理
- 执行日志记录

### 5. 邮件通知
- SMTP配置
- 模板化邮件内容
- 自动发送报告

## 使用方法

### 基础数据库检查
```bash
python db_checker.py
```

这将创建一个示例数据库并运行所有检查，最终生成一个名为 `database_check_report.txt` 的报告文件。

### 高级数据库检查
```bash
python db_checker_advanced.py
```

包含更多检查功能，会生成JSON格式的报告。

### 自动化检查
```bash
python db_checker_auto.py
```

无需交互即可运行检查。
